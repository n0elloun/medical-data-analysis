# Анализ эффективности терапии у госпитализированных пациентов

Учебный проект по анализу медицинских данных. Цель — оценить влияние новой монотерапии на исходы лечения пациентов со среднетяжёлым и тяжёлым течением болезни с использованием методологии подбора пар.

## Структура репозитория

```
medical-data-analysis/

│   ├── БД_с_моно_full.xlsx      # Пациенты, получавшие препарат
│   ├── БД_без_моно_full.xlsx    # Пациенты без препарата
│   ├── Показатель_D.xlsx        # Результаты анализа на D-димер
│   ├── Показатель_F.xlsx        # Результаты анализа на ферритин
│   ├── df_mono_known.xlsx       # Обработанная группа МОНО
│   ├── df_nomono_known.xlsx     # Обработанная группа БЕЗ МОНО
│   └── df_all.xlsx              # Объединённый датасет
```

## Данные

Исходные данные включают две группы пациентов со среднетяжёлым и тяжёлым течением болезни:

- **МОНО** — пациенты, которым вводился новый препарат
- **БЕЗ МОНО** — пациенты без новой терапии

По каждому пациенту доступны: номер истории болезни, даты госпитализации, пол, возраст, тяжесть течения, исход (выписан / умер / переведён), статус вакцинации, а также лабораторные показатели D-димер и ферритин из отдельных файлов.

## Методология

### Глава 1 — Предобработка

- Присвоение CaseID записям без идентификатора (новые ID не пересекаются с существующими)
- Удаление дубликатов: при повторных госпитализациях оставляется последняя
- Объединение таблиц через left join по CaseID
- Очистка числовых показателей (замена текстовых заглушек типа `>3000` на NaN)
- Фильтрация: оставляются только пациенты с известным исходом (умер / выписан)

### Глава 2 — Описательная статистика

Сравнение групп МОНО и БЕЗ МОНО по полу, возрасту и летальности до матчинга. Проводится односторонний z-тест на равенство летальности — показывается, что группы исходно несопоставимы (группа МОНО тяжелее), что обосновывает необходимость матчинга.

### Глава 3 — Матчинг и анализ исходов

Подбор пар методом greedy nearest-neighbor matching по критериям:

- одинаковый пол
- возраст ±3 года
- уровень D-димера ±10%
- уровень ферритина ±10%
- схожая тяжесть (среднетяжёлое / тяжёлое и крайне тяжёлое)

Дополнительно анализируется, как изменяется число пар при добавлении критерия одинакового статуса вакцинации.

После матчинга проводится:

- проверка сопоставимости групп (t-тест, критерий хи-квадрат)
- сравнение исходов жив/умер (хи-квадрат)
- сравнение сроков госпитализации у выживших (критерий Манна–Уитни)
- оценка влияния отдельных факторов на исход: возраст, D-димер, ферритин, пол, тяжесть, вакцинация

## Основные результаты

После подбора 1020 пар группы оказались полностью сопоставимы по всем критериям (p > 0.05 по всем факторам). Летальность в группе МОНО составила 15.4% против 17.9% в группе БЕЗ МОНО — тенденция в пользу препарата есть, однако различие статистически незначимо (p = 0.137). Сроки госпитализации у выживших в группе МОНО значимо выше (медиана 14 против 10 дней, p < 0.001), что объясняется более тяжёлым исходным состоянием пациентов и длительностью курса терапии. Значимыми предикторами смерти оказались возраст, уровни D-димера и ферритина, а также тяжесть заболевания.
